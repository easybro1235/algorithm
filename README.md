# 알고리즘 공부 일기장 (Algorithm Study Diary)

알고리즘 문제 해결 과정과 학습 내용을 기록하는 저장소입니다.

## 📚 목적 (Purpose)

- 알고리즘 문제 풀이 과정 기록
- 학습한 알고리즘과 자료구조 정리
- 문제 해결 패턴 및 접근법 학습
- 지속적인 성장 추적

## 📁 폴더 구조 (Folder Structure)

```
algorithm/
├── data-structures/     # 자료구조
├── sorting/            # 정렬 알고리즘
├── searching/          # 탐색 알고리즘
├── dynamic-programming/ # 동적 프로그래밍
├── greedy/             # 탐욕 알고리즘
├── graph/              # 그래프 알고리즘
├── string/             # 문자열 알고리즘
├── math/               # 수학 문제
└── etc/                # 기타
```

## 📝 작성 방법 (How to Use)

각 알고리즘 문제에 대해 다음 형식으로 기록합니다:

1. **문제 정보**: 문제 출처, 난이도, 링크
2. **문제 이해**: 문제 요약 및 핵심 포인트
3. **접근 방법**: 사용한 알고리즘/자료구조
4. **코드 구현**: 실제 구현 코드
5. **시간 복잡도**: Big-O 분석
6. **회고**: 배운 점, 개선점
