# 그래프 (Graph)

## 개념
그래프는 정점(Vertex)과 간선(Edge)으로 이루어진 자료구조입니다.

## 그래프 표현 방법
- 인접 행렬 (Adjacency Matrix)
- 인접 리스트 (Adjacency List)

## 주요 알고리즘
- **탐색**
  - DFS (깊이 우선 탐색)
  - BFS (너비 우선 탐색)
- **최단 경로**
  - 다익스트라 (Dijkstra) - 단일 시작점
  - 벨만-포드 (Bellman-Ford) - 음수 가중치
  - 플로이드-워셜 (Floyd-Warshall) - 모든 쌍
- **최소 신장 트리 (MST)**
  - 크루스칼 (Kruskal)
  - 프림 (Prim)
- **위상 정렬** (Topological Sort)
- **강한 연결 요소** (SCC)
- **이분 매칭** (Bipartite Matching)
