# 수학 (Math)

## 개념
알고리즘 문제 해결에 필요한 수학적 개념과 기법입니다.

## 주요 주제
- **정수론**
  - 최대공약수 (GCD)
  - 최소공배수 (LCM)
  - 소수 판별
  - 에라토스테네스의 체
  - 모듈러 연산
- **조합론**
  - 순열 (Permutation)
  - 조합 (Combination)
  - 파스칼의 삼각형
  - 카탈란 수
- **기하학**
  - 벡터 연산
  - CCW (Counter-Clockwise)
  - 볼록 껍질 (Convex Hull)
- **기타**
  - 비트 연산
  - 분할 정복을 이용한 거듭제곱
