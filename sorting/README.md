# 정렬 (Sorting)

## 개념
정렬은 데이터를 특정 순서로 배열하는 알고리즘입니다.

## 주요 정렬 알고리즘
- 버블 정렬 (Bubble Sort) - O(n²)
- 선택 정렬 (Selection Sort) - O(n²)
- 삽입 정렬 (Insertion Sort) - O(n²)
- 병합 정렬 (Merge Sort) - O(n log n)
- 퀵 정렬 (Quick Sort) - O(n log n) 평균
- 힙 정렬 (Heap Sort) - O(n log n)
- 계수 정렬 (Counting Sort) - O(n + k)
- 기수 정렬 (Radix Sort) - O(d * (n + k))
